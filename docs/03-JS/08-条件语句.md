# 条件语句

if ... else 

## 比较运算符的返回
任何不是 false, undefined, null, 0, NaN 的值，或一个空字符串（''）在作为条件语句进行测试时实际返回true。

## 逻辑运算符
- && 
- ||
- ！

## switch语句
```
switch (expression) {
  case choice1:
    run this code
    break;

  case choice2:
    run this code instead
    break;
    
  // include as many cases as you like

  default:
    actually, just run this code
}
```

## 三元运算符
( condition ) ? run this code : run this code instead
