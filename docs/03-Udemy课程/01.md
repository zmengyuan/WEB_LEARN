**Function declarations vs. expressions**

```js
//function declaration
function calcAge1(birthYear){
    return 2037-birthYear;
}
//function expression
const calcAge2 = function (birthYear) {
    return 2037 - birthYear;
}
//arrow function
const calAge3 = birthYear => 2037 - birthYear;
```
